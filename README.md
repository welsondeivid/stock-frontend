# Estoque Front-end

Front-end do sistema de controle de estoque,
desenvolvido com React,
consumindo uma API REST construída em Java com Quarkus.

### Tecnologias usadas
* JavaScript
* React
* Vite
* Axios
* React Router DOM
* HTML5
* CSS3

### Como rodar o projeto

```bash
npm install
npm run dev
```
A aplicação sobe em:<br>
Front-end: http://localhost:5173

Backend

Este projeto consome a API REST disponível em:<br>
https://github.com/welsondeivid/stock-backend
